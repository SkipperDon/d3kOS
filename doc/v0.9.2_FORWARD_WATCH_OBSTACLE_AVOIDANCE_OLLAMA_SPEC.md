## MISSION STATEMENT

Implement Forward Watch — AI-powered obstacle detection on the bow camera:
- Detect boats, kayaks, buoys, logs, debris, docks, and people in water
- Alert crew at 3 severity levels (critical/warning/info)
- Overlay detection boxes on the marine-vision.html single view
- Send alerts to SignalK for dashboard integration
- Package as a standalone SignalK plugin for community distribution

**Scope:**
- Detection runs on bow camera ONLY (cam01)
- Detection triggered manually OR automatically when speed > 1 knot
- Detection interval: every 3 seconds (not continuous — Pi resources)
- Alerts: visual overlay + audio beep + SignalK notification

## PHASE 2: FORWARD WATCH DETECTOR

**File:** `/opt/d3kos/services/marine-vision/forward_watch.py`

```python
#!/usr/bin/env python3
"""
d3kOS Forward Watch — Obstacle Avoidance System
Version: 0.9.2
Detects marine hazards in bow camera feed using YOLOv8.

Alert levels:
  CRITICAL: High-confidence object, potential collision risk
  WARNING:  Object detected, monitor situation
  INFO:     Object in scene, low priority
"""
import json
import os
import subprocess
import time
import threading
import logging
from datetime import datetime
from typing import Optional

from flask import Flask, jsonify, request
from flask_cors import CORS

# Configure logging
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s [ForwardWatch] %(levelname)s: %(message)s'
)
logger = logging.getLogger('forward_watch')

app = Flask(__name__)
CORS(app)

RESULTS_FILE = '/tmp/d3kos_forward_watch.json'
MODEL_PATH = '/opt/d3kos/models/yolov8n.pt'
BOW_CAMERA_RTSP = 'rtsp://admin:d3kos2026@10.42.0.100:554/h264Preview_01_main'
FRAME_TEMP = '/tmp/d3kos_fw_frame.jpg'
DETECTION_INTERVAL = 3  # seconds between detections

# COCO classes that are relevant to marine obstacle detection
MARINE_COCO_CLASSES = {
    0: 'person',     # Person in water — CRITICAL
    8: 'boat',       # Sailboat proxy
    2: 'car',        # NOT relevant
    9: 'traffic light',  # Buoy proxy
    56: 'chair',     # Debris proxy
    73: 'book',      # Not relevant
}

# Class to alert level mapping
CLASS_ALERT_LEVELS = {
    'person': 'CRITICAL',
    'boat': 'WARNING',
    'sailboat': 'WARNING',
    'kayak': 'WARNING',
    'buoy': 'INFO',
    'log': 'WARNING',
    'debris': 'INFO',
    'dock': 'INFO',
}

# Detection state
detection_active = False
detection_thread: Optional[threading.Thread] = None
last_results = {
    'timestamp': None,
    'detections': [],
    'alert_level': 'NONE',
    'frame_path': None,
    'error': None
}


def capture_frame() -> Optional[str]:
    """Capture single frame from bow camera."""
    try:
        result = subprocess.run(
            ['ffmpeg', '-y', '-rtsp_transport', 'tcp',
             '-i', BOW_CAMERA_RTSP,
             '-frames:v', '1', '-q:v', '2', FRAME_TEMP],
            capture_output=True, timeout=10
        )
        if result.returncode == 0 and os.path.exists(FRAME_TEMP):
            return FRAME_TEMP
    except (subprocess.TimeoutExpired, Exception) as e:
        logger.error(f'Frame capture failed: {e}')
    return None


def run_detection(frame_path: str) -> list:
    """Run YOLOv8 inference on frame. Returns list of detection dicts."""
    if not os.path.exists(MODEL_PATH):
        logger.warning('YOLOv8 model not found, using mock detections')
        return []
    try:
        from ultralytics import YOLO
        model = YOLO(MODEL_PATH)
        results = model(frame_path, verbose=False, conf=0.4, device='cpu')
        detections = []
        for r in results:
            for box in r.boxes:
                cls_id = int(box.cls[0])
                cls_name = model.names[cls_id].lower()
                conf = float(box.conf[0])
                xyxy = box.xyxy[0].tolist()
                # Map COCO class to marine concept
                marine_name = map_to_marine(cls_name)
                alert = CLASS_ALERT_LEVELS.get(marine_name, 'INFO')
                detections.append({
                    'class': marine_name,
                    'coco_class': cls_name,
                    'confidence': round(conf, 3),
                    'alert_level': alert,
                    'bbox': [round(x, 1) for x in xyxy],
                    'detected_at': datetime.now().isoformat()
                })
        return detections
    except Exception as e:
        logger.error(f'Detection failed: {e}')
        return []


def map_to_marine(coco_class: str) -> str:
    """Map COCO class name to marine object category."""
    mapping = {
        'person': 'person',
        'boat': 'boat',
        'surfboard': 'kayak',
        'skateboard': 'debris',
        'sports ball': 'buoy',
        'umbrella': 'buoy',
        'clock': 'buoy',
        'traffic light': 'buoy',
        'car': 'boat',
        'truck': 'boat',
    }
    return mapping.get(coco_class, coco_class)


def get_overall_alert(detections: list) -> str:
    if not detections:
        return 'NONE'
    levels = {'NONE': 0, 'INFO': 1, 'WARNING': 2, 'CRITICAL': 3}
    max_level = max(levels.get(d['alert_level'], 0) for d in detections)
    return {0: 'NONE', 1: 'INFO', 2: 'WARNING', 3: 'CRITICAL'}[max_level]


def write_results(detections: list, error: str = None):
    global last_results
    alert = get_overall_alert(detections)
    last_results = {
        'timestamp': datetime.now().isoformat(),
        'detections': detections,
        'detection_count': len(detections),
        'alert_level': alert,
        'frame_path': FRAME_TEMP if os.path.exists(FRAME_TEMP) else None,
        'error': error
    }
    with open(RESULTS_FILE, 'w') as f:
        json.dump(last_results, f, indent=2)
    if alert in ('WARNING', 'CRITICAL'):
        logger.warning(f'FORWARD WATCH ALERT [{alert}]: {len(detections)} objects detected')
    elif detections:
        logger.info(f'Forward Watch: {len(detections)} objects, level: {alert}')


def detection_loop():
    """Background thread: capture and detect every DETECTION_INTERVAL seconds."""
    global detection_active
    logger.info('Forward Watch detection loop started')
    while detection_active:
        try:
            frame_path = capture_frame()
            if frame_path:
                detections = run_detection(frame_path)
                write_results(detections)
            else:
                write_results([], error='Camera unavailable')
        except Exception as e:
            write_results([], error=str(e))
        time.sleep(DETECTION_INTERVAL)
    logger.info('Forward Watch detection loop stopped')


# --- API ENDPOINTS ---

@app.route('/forward-watch/start', methods=['POST'])
def start_detection():
    global detection_active, detection_thread
    if detection_active:
        return jsonify({'status': 'already_running'})
    detection_active = True
    detection_thread = threading.Thread(target=detection_loop, daemon=True)
    detection_thread.start()
    return jsonify({'status': 'started', 'interval_seconds': DETECTION_INTERVAL})


@app.route('/forward-watch/stop', methods=['POST'])
def stop_detection():
    global detection_active
    detection_active = False
    return jsonify({'status': 'stopped'})


@app.route('/forward-watch/status', methods=['GET'])
def get_status():
    return jsonify({
        'active': detection_active,
        'interval_seconds': DETECTION_INTERVAL,
        'model_available': os.path.exists(MODEL_PATH),
        'camera_ip': '10.42.0.100'
    })


@app.route('/forward-watch/results', methods=['GET'])
def get_results():
    try:
        with open(RESULTS_FILE) as f:
            return jsonify(json.load(f))
    except FileNotFoundError:
        return jsonify({
            'timestamp': None,
            'detections': [],
            'detection_count': 0,
            'alert_level': 'NONE',
            'error': 'No detections yet'
        })


@app.route('/forward-watch/detect-once', methods=['POST'])
def detect_once():
    """Run single detection (for testing)."""
    frame_path = capture_frame()
    if not frame_path:
        return jsonify({'error': 'Could not capture frame from bow camera'}), 503
    detections = run_detection(frame_path)
    write_results(detections)
    return jsonify(last_results)


@app.route('/forward-watch/frame', methods=['GET'])
def get_annotated_frame():
    """Return latest frame (with or without annotations)."""
    from flask import send_file
    if os.path.exists(FRAME_TEMP):
        return send_file(FRAME_TEMP, mimetype='image/jpeg')
    return jsonify({'error': 'No frame available'}), 404


@app.route('/health', methods=['GET'])
def health():
    return jsonify({
        'status': 'ok',
        'service': 'forward-watch',
        'version': '0.9.2',
        'detection_active': detection_active,
        'model_loaded': os.path.exists(MODEL_PATH)
    })


if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8085, debug=False)
```

**File:** `index.js`
```javascript
'use strict';
const fs = require('fs');

const RESULTS_FILE = '/tmp/d3kos_forward_watch.json';
const CHECK_INTERVAL_MS = 3000;

module.exports = function(app) {
  const plugin = {
    id: 'forward-watch-plugin',
    name: 'Forward Watch',
    description: 'Reads Forward Watch detection results and sends to SignalK'
  };

  let pollInterval = null;
  let lastTimestamp = null;

  plugin.start = function(options) {
    app.debug('Forward Watch plugin starting');
    pollInterval = setInterval(checkResults, CHECK_INTERVAL_MS);
  };

  plugin.stop = function() {
    if (pollInterval) clearInterval(pollInterval);
    app.debug('Forward Watch plugin stopped');
  };

  function checkResults() {
    try {
      if (!fs.existsSync(RESULTS_FILE)) return;
      const data = JSON.parse(fs.readFileSync(RESULTS_FILE, 'utf8'));

      // Only process new results
      if (data.timestamp === lastTimestamp) return;
      lastTimestamp = data.timestamp;

      const alert = data.alert_level || 'NONE';
      if (alert === 'NONE' || !data.detections || data.detections.length === 0) return;

      // Send SignalK notification
      const highestDetection = data.detections.reduce((prev, curr) => {
        const levels = { 'INFO': 1, 'WARNING': 2, 'CRITICAL': 3 };
        return (levels[curr.alert_level] || 0) > (levels[prev.alert_level] || 0) ? curr : prev;
      }, data.detections[0]);

      const skState = {
        'INFO': 'normal',
        'WARNING': 'warning',
        'CRITICAL': 'alarm'
      }[alert] || 'normal';

      const notification = {
        message: `Forward Watch: ${highestDetection.class} detected (${alert})`,
        state: skState,
        method: alert === 'CRITICAL' ? ['sound', 'visual'] : ['visual'],
        timestamp: data.timestamp
      };

      app.handleMessage(plugin.id, {
        updates: [{
          source: { label: plugin.id },
          values: [
            {
              path: 'notifications.forwardWatch',
              value: notification
            },
            {
              path: 'environment.forwardWatch.alertLevel',
              value: alert
            },
            {
              path: 'environment.forwardWatch.detectionCount',
              value: data.detection_count || 0
            }
          ]
        }]
      });

      app.debug(`Forward Watch: ${alert} — ${JSON.stringify(highestDetection)}`);
    } catch (e) {
      app.error('Forward Watch check failed: ' + e.message);
    }
  }

  plugin.schema = {
    title: 'Forward Watch Configuration',
    type: 'object',
    properties: {
      enabled: {
        type: 'boolean',
        title: 'Enable Forward Watch',
        default: true
      }
    }
  };

  return plugin;
};
```

```markdown
# d3kOS Forward Watch — User Guide

## What It Does
Monitors bow camera for obstacles: boats, kayaks, people in water,
logs, buoys, and debris. Alerts you before potential collisions.

## Alert Levels
- INFO (green): Object in scene, low risk
- WARNING (orange): Object detected, stay alert
- CRITICAL (red): Potential collision hazard — take action

## How to Use
1. Open Marine Vision: http://d3kos.local/marine-vision.html
2. Ensure Bow camera is selected
3. Click "Start Forward Watch"
4. Watch for alert banners while underway
5. Click "Detect Once" for an immediate snapshot check

## Automatic Mode (future)
Forward Watch will auto-start when speed > 1 knot (coming in v0.9.3)

## Performance
Detection runs every 3 seconds.
Do not rely solely on Forward Watch for navigation.
Always maintain a proper watch by eye.
```
