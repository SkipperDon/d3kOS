ℹ️ About d3kOS

d3kOS is a comprehensive marine electronics system for engine monitoring, trip logging, and performance analysis.

Built on Raspberry Pi 4B with Signal K integration, NMEA2000 support, and a touch-friendly dark interface optimized for 10.1" marine displays.

🤖 AI Assistant — Gemini API

What is Gemini?
Google Gemini is an AI that gives Helm smarter, conversational answers to complex questions — engine diagnostics, navigation advice, marine regulations, and more.

How to get a free API key (2 minutes):

  1. Open a browser and go to aistudio.google.com
  2. Sign in with your Google account
  3. Click "Get API key" in the left sidebar
  4. Click "Create API key"
  5. Copy the key — it starts with AIza...
  6. Paste it in the field below and click Save Configuration

✓ Free tier is sufficient  |  ✓ Key stays on your Pi, never sent to any other server  |  ✓ Works offline for engine/sensor queries
