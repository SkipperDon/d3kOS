✅ When to Use Simulator

  • Indoor Development: Test system without running engine
  • UI Testing: Verify dashboard displays and alarms
  • Demos: Show system capabilities to others
  • Training: Learn system without burning fuel
  • Edge Cases: Test high RPM, failures, etc.

🔴 Critical Risks of Leaving Simulator ON

⚠️
Real Engine Data Ignored: If CX5106 is connected, actual RPM, boost, and trim values will be overridden by simulated data. You won't see real engine conditions.
🚨
False/Missing Alarms: Real engine problems (overheating, low oil) won't trigger alerts because simulated values are always "normal".
❌
User Confusion: Easy to forget simulator is active and make operational decisions based on fake data.
🔧
Troubleshooting Issues: Diagnosing real problems becomes difficult when data is simulated.
📡
Network Pollution: Simulated data broadcast on NMEA2000 network could confuse chartplotters and other displays.

🛡️ Built-in Safety Features

  • Auto-Disable on Boot: Simulator is OFF by default after reboot
  • Dashboard Banner: Orange warning shown on all pages when active
  • Manual Enable Required: Must be manually turned on each session
  • Visual Indicators: Simulated values displayed in orange
