✓ Solution: Install o-charts manually (see below)

📦 O-Charts Manual Installation for ARM64

Step 1: Purchase Charts

  • Visit o-charts.org
  • Purchase charts for your region (e.g., US, Canada, Europe)
  • You'll receive a license key via email

Step 2: Download ARM64 Plugin

  • Login to your o-charts account
  • Go to: My oeSENC Charts → Downloads
  • Download: Linux ARM64 version (NOT x86_64, NOT Android)
  • File will be named: oesenc_pi-X.X.X-linux-arm64.tar.gz

Step 3: Install Plugin

Transfer file to Pi and run:

# Extract plugin to OpenCPN directory
cd ~/.opencpn/plugins/
tar -xzf ~/Downloads/oesenc_pi-*-linux-arm64.tar.gz

# Restart OpenCPN

Step 4: Activate License

  • Launch OpenCPN (button above)
  • Go to: Tools → Options → Plugins
  • Find "oeSENC" plugin → Enable
  • Enter your license key from o-charts.org
  • Download your purchased charts

🆓 Free Chart Alternatives (No Plugin Required)

  • Built-in Charts: OpenCPN includes basic world charts
  • NOAA ENC (US): Download from charts.noaa.gov
  • CHS (Canada): Download from Canadian Hydrographic Service
  • OpenSeaMap: Community-contributed free charts

These can be manually downloaded and loaded into OpenCPN without plugins.
