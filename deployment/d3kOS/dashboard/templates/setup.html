🔅 DIP Switch Settings — Starboard Engine
ROW 2 — Regional
Actisense EMU-1
DIP switches on the EMU-1 are set using the Actisense NMEA Reader software on a PC. Refer to the EMU-1 Quick Reference Card included in the box. Key settings: select your cylinder count, set the engine instance (0 = single/port, 1 = starboard), and match the tank sender standard for your region.
NMEA 2000 Gateway
Refer to the configuration guide included with your gateway. Key parameters to set: engine instance number (0 for single/port, 1 for starboard), cylinder count, and fuel/level sender resistance range for your region.
