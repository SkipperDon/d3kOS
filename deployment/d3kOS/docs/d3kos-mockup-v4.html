🚀 Getting Started — How to Use d3kOS
Daily Use
1
Pi boots → d3kOS dashboard auto-launches at localhost:3000
2
Charts → Main Menu → AvNav Charts → live GPS at localhost:8080
3
Weather → Dashboard → 🌤 Weather → Windy / RainViewer panels open
4
AI Nav → Main Menu → AI Navigation → marine queries at localhost:3001
5
Offline → AvNav + GPS work without internet. AI falls back to Ollama at 192.168.1.36:11434
Emergency Procedures
!
Dashboard down → sudo systemctl restart d3kos-dashboard
!
No GPS in AvNav → Check Signal K at localhost:8099 → Data Browser
!
AvNav down → Use OpenCPN Fallback from d3kOS menu — fully offline
!
AI Proxy down → sudo systemctl restart d3kos-gemini
⚠
AI not responding → Check Ollama at 192.168.1.36:11434 on LAN

ℹ️ About d3kOS

d3kOS is a comprehensive marine electronics system for chart navigation, engine monitoring, AI-assisted passage planning, and vessel camera monitoring.

Built on Raspberry Pi 4B with AvNav (primary charts at :8080), Signal K integration (:8099), NMEA support via OpenPlotter (:8081), and Gemini AI (:3001) with Ollama offline fallback (192.168.1.36:11434).
